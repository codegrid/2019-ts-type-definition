TOC Module Library

はじめに

これは「自分で書く、型定義ファイル」のデモページです。

使い方

ライブラリのインポート

ES modules 形式で実装されているため、次のようにライブラリ本体と型を import します。

import Toc, { TocList } from './lib/toc';

目次情報の取得

Toc.getTocList() メソッドを使用し次のように取得できます。

getTocList

const target = document.querySelector('article') as HTMLElement;
const tocList: TocList = Toc.getTocList(target);

result

目次のマークアップの取得

Toc.getTocHtml() メソッドを使用し次のように取得できます。

getTocHtml

const target = document.querySelector('article') as HTMLElement;
const tocHtml: string = Toc.getTocHtml(target);

result
